Overview

Properties are persistent entities that are persisted between program invocations. There are Property subclasses for all the JavaFX basic types (excluding Duration) as well as Sequences of basic types.

By default properties bound to a Configuration are only persisted when the user clicks "Save" in the dialog. However, if a Property has "autoSave" enabled it will be persisted any time its state changes, which allows application-driven state to be saved in addition to user state.
